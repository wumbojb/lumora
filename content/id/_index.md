## ğŸš€ Installation

Add Lumora as a submodule to your Hugo site:

```bash
git submodule add https://github.com/masputrawae/lumora themes/lumora
````

Then update your `config.yaml` or `config.toml` to use the theme:

```yaml
theme: lumora
```
